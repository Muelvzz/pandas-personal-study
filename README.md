# Pandas Learning Notebook

This project contains a beginner-friendly Jupyter notebook, main.ipynb, designed to introduce the core concepts of pandas through practical examples.

This is my personal learnings after taking the pandas documentations. Starting with the "What kind of data does pandas handle?" into "How to combine data from multiple value?"

The notebook walks through common data analysis tasks using sample datasets stored in the data folder. It covers topics such as reading CSV files, inspecting and selecting data, filtering rows, creating plots, deriving new columns, calculating summary statistics, sorting data, and combining tables.

## What you will learn

- How to load and explore tabular data with pandas
- How to select columns and filter rows
- How to create new columns from existing data
- How to generate simple visualizations with matplotlib
- How to summarize and reshape data

## Project files

- main.ipynb: the main notebook with the examples
- data: sample datasets used throughout the notebook
- pyproject.toml: project dependencies and configuration

## Getting started

Open main.ipynb in Jupyter Notebook or VS Code and run the cells in order.

This notebook is intended as a hands-on introduction to pandas for learners who want to practice real data manipulation and analysis workflows.
